Anotações e Asserções

Domine a API do Petal Test com este guia completo de anotações para controle de testes e asserções para verificação de resultados.

Anotações

As anotações permitem configurar o comportamento dos seus testes e controlar o ciclo de vida da execução.

Anotação Descrição Exemplo
@ClassTestable() Marca uma classe como contendo métodos de teste @ClassTestable() class MeusTestes { ... }
@Test() Marca um método como um teste @Test() void testeExemplo() { ... }
@BeforeEach() Executado antes de cada método de teste @BeforeEach() void setUp() { ... }
@AfterEach() Executado após cada método de teste @AfterEach() void tearDown() { ... }
@RepeatedTest(value: n) Executa o teste 'n' vezes @RepeatedTest(value: 3) void teste() { ... }
@ParameterizedTest() Marca um teste como parametrizado @ParameterizedTest() @ValueSource(...) void teste() { ... }
@DisplayName(displayName: "texto") Define um nome amigável para o teste @DisplayName(displayName: "Teste de login")

Exemplo Completo

Veja como usar várias anotações juntas em uma classe de teste:

user_service_test.dart
@ClassTestable()
class UserServiceTest {
  late UserService userService;
  
  @BeforeEach()
  void setUp() {
    userService = UserService(MockUserRepository());
  }
  
  @Test()
  @DisplayName(displayName: "Cria usuário com email válido")
  void testCreateUser_validEmail() {
    final user = userService.createUser('test@example.com');
    expect(user.email, equals('test@example.com'));
  }
  
  @RepeatedTest(value: 5)
  void testCreateUser_randomEmail() {
    final email = 'test${Random().nextInt(100)}@example.com';
    final user = userService.createUser(email);
    expect(user.email, contains('@example.com'));
  }
  
  @AfterEach()
  void tearDown() {
    userService.dispose();
  }
}

Asserções

O Petal Test fornece uma variedade de asserções para verificar seus resultados:

Asserções Básicas

// Igualdade
expect(result, equals(expected));
expect(result, isNot(unexpected));

// Booleanos
expect(result, isTrue);
expect(result, isFalse);
expect(result, isNull);
expect(result, isNotNull);

// Números
expect(result, greaterThan(min));
expect(result, lessThan(max));
expect(result, inRange(min, max));

Asserções para Coleções

expect(list, contains(value));
expect(list, hasLength(length));
expect(map, containsValue(value));
expect(list, isNotEmpty);
